# Pipex

This project that aims to recreate the behavior of shell piping using processes, file descriptors, and UNIX pipes.
The program must reproduce the following shell command:

```bash
< infile cmd1 | cmd2 > outfile
```

## Installation

When you have Git cloned the repository, to compute you need to make the program.

## Use

When you have compute the program, you just need to give him: infile "cmd1" "cmd2" outifle.
